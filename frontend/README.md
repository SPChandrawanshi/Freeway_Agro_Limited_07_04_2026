# Freeway Agro Limited - Website

A modern, responsive, production-ready website for Freeway Agro Limited built with Next.js 16, React 19, and Tailwind CSS.

## Features

- 🌿 7 Complete Pages (Home, About, Products, Environment, FAQ, Blog, Contact)
- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- ✨ Advanced Animations and Transitions
- 🎨 Professional Design System with Custom Colors
- 📧 Contact Form with Validation
- 🔍 SEO Optimized
- ⚡ Fast Performance with Next.js Turbopack

## Pages

1. **Home** - Hero section with product overview and sustainability commitment
2. **About** - Company story, values, mission, and vision
3. **Products** - Detailed product information and specifications
4. **Environment** - Environmental sustainability initiatives
5. **FAQ** - Frequently asked questions
6. **Blog** - Latest news and insights
7. **Contact** - Contact form and partnership opportunities

## Tech Stack

- **Framework:** Next.js 16.2.2 (with Turbopack)
- **React:** 19.2.4
- **Styling:** Tailwind CSS v4
- **Language:** JavaScript/JSX
- **Image Optimization:** Next.js Image Component

## Color System

- Primary: #0F3D3E (Dark Green)
- Secondary: #F4A261 (Orange)
- Accent: #2A9D8F (Light Green)

## Installation

```bash
cd frontend
npm install
npm run dev
```

The website will be available at `http://localhost:3000`
